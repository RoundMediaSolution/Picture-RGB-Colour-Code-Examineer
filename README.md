# Picture-RGB-Colour-Code-Examineer
The Picture RGB &amp; Color Code Examiner is a high-precision Python utility designed to run seamlessly in Google Colab. It transforms any uploaded image into a comprehensive dataset by extracting the exact RGB (Red, Green, Blue) values and Hexadecimal color codes for every single pixel.

This tool is ideal for image analysis, custom palette generation, and preparing visual data for machine learning or automation workflows.Core FeaturesPixel-Perfect Extraction: Maps every $(x, y)$ coordinate of an image to its specific color data.Structured Output: Automatically generates a downloadable CSV file containing:X & Y CoordinatesRed Value ($0-255$)Green Value ($0-255$)Blue Value ($0-255$)Hex Color Code (e.g., #FFFFFF)Cloud-Based Processing: Utilizes Google Colab’s infrastructure to handle high-resolution images without taxing your local CPU/RAM.Minimalist Workflow: Simple "Upload & Run" logic using PIL (Pillow) and Pandas libraries.Technical WorkflowImage Input: The user uploads a file (.jpg, .png, etc.) directly to the Colab runtime.Processing: The script iterates through the image matrix using numpy for high-speed array manipulation.Data Conversion: RGB tuples are converted to Hex strings via bit-shifting logic.Export: A .csv file is compiled and prompted for immediate download.
