<h2>CSV Data to Image Reconstructor</h2>
<p>The CSV Data to Image Reconstructor is a specialized Python utility designed for Google Colab. 
<br>It performs "reverse pixel mapping" by taking structured RGB/Hex data from a CSV file and rebuilding the original visual image.
<br>This tool is particularly useful for verifying data integrity, experimenting with algorithmic image manipulation, or visualizing large-scale color datasets.

**Core FeaturesInstant Reconstruction:**
<ol>
  <li>Reads $(X, Y)$ coordinates and color codes from a CSV to redraw a pixel-perfect image.
  <li>Format Flexibility: Supports both RGB Triplet (Red, Green, Blue columns) and Hexadecimal input formats.
  <li>Auto-Dimensioning: Intelligently determines the image width and height based on the maximum coordinates found in the dataset.
  <li>High-Performance Rendering: Uses numpy arrays and PIL (Pillow) to ensure fast image generation, even for high-resolution datasets.</ol>

**Technical WorkflowData Ingestion:** 
<ol>
  <li></li>The user uploads the CSV file generated by the Color Code Examiner.
  <li>Coordinate Mapping: The script identifies the maximum $X$ and $Y$ values to create a blank canvas of the correct size.
  <li>Pixel Injection: It loops through each row of the CSV, injecting the specific color values into the corresponding $(x, y)$ location in a 3D array.
  <li>Image Synthesis: The array is converted into an Image object and displayed/saved as a .png or .jpg.</ol>
